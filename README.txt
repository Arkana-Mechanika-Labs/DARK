2026

WHAT THIS IS

DARK is a desktop workbench for inspecting, editing, validating, and researching
Darklands data files.

Current scope includes:
- save files
- world data (cities, locations, descriptions, items, saints, formulae, enemies)
- dialog cards and message archives
- PIC and IMC graphics workflows
- PAN presentation sequence playback
- font editing
- CAT archive browsing and rebuilding
- validation, coverage, and research-oriented tools


IMPORTANT DISCLAIMER

This software is provided as an experimental reverse-engineering and modding tool.
It is not affiliated with the original creators or rights holders of Darklands.

Use it at your own risk.

Some parts of DARK are mature enough for practical use, but some parts remain
experimental, incomplete, or still under active research. Certain file families
may be view-only, partially understood, or not yet safe for routine editing.

Always assume that mistakes, incomplete format knowledge, or unexpected bugs are
still possible.


BACK UP YOUR FILES

Before editing anything:

1. Make a full backup copy of your Darklands game directory.
2. Prefer working on a duplicate/modding copy, not your only clean install.
3. Keep extra backups before making major changes.

DARK creates backup files in some workflows, but you should NOT rely on that as
your only protection.

Your own manual backup remains essential.


SYSTEM REQUIREMENTS

- Windows 64-bit
- A Darklands game data directory

No separate Python installation should be required for the packaged executable.


USING DARK

1. Start DARK.exe
2. Set the DL Data Path to your Darklands folder
3. Choose a tool from the sidebar
4. Inspect or edit the data you want
5. Save carefully and test your changes in-game


GOOD PRACTICE

- Edit a copy of the game files
- Change a small number of things at a time
- Test frequently
- Keep notes about what you changed
- Use validation and coverage tools when available


EXPERIMENTAL AREAS

Some areas are still works in progress, including research-oriented support for
formats such as IMG and PAN, and other less fully understood files.

Even in supported editors, some edge cases may still exist.


NO GAME DATA INCLUDED

This package does not include Darklands game data.
You must supply your own installation or working copy.
